For the first 48 hours (until 12th September) please only sign up to this game if you have previously been part of the Cogflower Caravan on The Gauntlet.

Ultraviolet Grasslands is a psychedelic metal setting by Luka Rejec about exploration, survival, drugs and resource management.
This is a continuation of the adventures of the Cogflower Caravan on her journey to the Black City.
We will be using Whitehack, a rules lite version of the world’s most popular RPG.
This event is scheduled to run for 3 hours. It will be recorded assuming all parties agree.
There will be 3 sessions in November. Being able to attend each session is ideal but not required. You must RSVP to each session you wish to attend.
This game will follow the Gauntlet Community Code of Conduct: https://www.gauntlet-rpg.com/community-code-of-conduct.html. We will be using the X-Card and other safety tools players suggest.
