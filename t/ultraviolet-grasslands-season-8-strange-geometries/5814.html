This is a continuation of the adventures of the Cogflower Caravan on her journey to the Black City.

There will be 4 sessions in October. Being able to attend each session is ideal but not required. You must RSVP to each session you wish to attend.
This game will follow the Gauntlet Community Code of Conduct: https://www.gauntlet-rpg.com/community-code-of-conduct.html. We will be using the X-Card and other safety tools players suggest.
