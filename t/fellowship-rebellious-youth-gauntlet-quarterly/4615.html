This is the first four sessions in what will be a quarterly run of Fellowship by Jacob Randolph, using the In Rebellion supplement. Fellowship is a PbtA game in which you play fantasy heroes on a quest against evil, where the friendships you make and bonds you forge can be stronger than steel. Sessions will be three hours in length. We will probably play using the Jitsi browser-based chat tool. This game will be recorded, and may be released on Youtube with the consent of all participants.

This game will be played using the open table policy, so you are free to sign up for as many or few sessions as able. Signing up for all sessions is encouraged, but is not required. We are likely to be running an extended session zero for the first session, though, so I would advise against only signing up for the first session. This is a quarterly series and will continue at the same time in May and June, though those sessions will be posted at a later date. (Advance notice may be given to continuing players).

I will be running this game using the In Rebellion supplement, in which the Fellowship live in a world where the forces of evil have won, and are ruling with an iron fist. The Fellowship will be young, plucky heroes leading a rebellion against the Empire. I will also be running this game with a couple of minor hacks from the main game. I want to avoid using “fantasy races”, so some of the playbooks will be tweaked or curated. Secondly, I will be running this game as “children’s fantasy” - so think Avatar, She-Ra, or The Dragon Prince. Aside from the tonal considerations and age of the protagonists, I will be tweaking a couple of rules to support this theme.

Content Warning: As children’s fantasy, this game should have roughly a “PG-13” content rating. While there will be no graphic sexual or violent content, this is still a game about resisting an authoritarian regime. Due to the age of the protagonists, the game will also by its nature feature children in peril. Though ultimately it is a game about hope, and one that de-emphasises violent solutions to problems. As a particular consideration, the Nemesis playbook will be available which allows a player to take an antagonist role in the game, and as such will introduce a “PvP” element - we can discuss this role and if someone wishes to take it on in our session zero. We will have a discussion on tone and content before the game begins, as well as establishing Lines & Veils to provide a comfortable play experience. This game will take place under the Gauntlet Community Code of Conduct and the safety tools Script Change and X-Card will be in use.
