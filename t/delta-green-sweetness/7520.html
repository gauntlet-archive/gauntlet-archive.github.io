Sweetness is a scenario is written by Dennis Detwiller. Delta Green is a horror roleplaying game. There might be violence and gore. Sweetness specifically has themes of mental illness and child abuse.

We will be using the X-Card, Lines and Veils and CATS. The game will follow the Gauntlet Community Code of Conduct. We will be using streamyard. The session will be recorded but only made public if all players agree. I will send a reminder per email a week before we play.

This is part of the Gauntlet Delta Green shared world. New and returning agents are equally welcome. Surviving agents new to the program will be added to the roster at the end of the series.
