Custom Blades Playbook Challenge

Game Design
Dissonance September 8, 2020, 4:43pm #1

Over the weekend I put together some Blades Playbook Templates for Affinity Publisher. These will help you to create attractive custom playbooks, ala The Sleuth!

Each template also contains some helpful advice from me on designing quality playbooks for use in the base game.

I feel there is potential for a lot more custom playbooks for the game and would love to see yours so I’m making it a challenge!

This challenge is for the base game so feel free to tie yours into Doskvol and the larger world of Blades. Then post them here for feedback or just to share!
